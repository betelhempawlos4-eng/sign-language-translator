Train Your Own Model (Option A)

Use Google Teachable Machine to train a custom gesture recogniser with your own hand — no code needed. Takes about 20 minutes.

1Go to teachablemachine.withgoogle.com → New Project → Pose Project
2Create a class for each sign (A, B, C…). Record 30–50 samples each using your webcam.
3Click Train Model and wait ~2 minutes.
4Click Export Model → TensorFlow.js → Upload (shareable link) and copy the URL.

Production Backend (Option C)

For full word-level recognition across all languages, you need a Python backend. Here is the architecture:

1Dataset: Download WLASL (2,000 ASL words) from github.com/dxli94/WLASL
2Extract: Run MediaPipe on each video frame to get landmark sequences (63 values × N frames)
3Train: LSTM or Transformer model on landmark sequences → sign label
4Serve: Flask API on Render → this app sends landmarks, API returns sign
5Deploy: render.com → Web Service → connect GitHub → auto-deploy

Detection pipeline

MediaPipe Hands → 21 hand landmarks → normalise coordinates → rule-based + TF.js classifier → sign label with confidence scores

Why 21 landmarks?

5 fingers × 4 joints (MCP+PIP+DIP+Tip) = 20 points + 1 wrist = 21 total. Each gives (x,y,z) → 63 numbers per frame. That's all the information needed to identify any hand pose.

Coordinate system

X = horizontal (0=left, 1=right) · Y = vertical (0=top, 1=bottom) · Z = depth (negative = closer to camera)
