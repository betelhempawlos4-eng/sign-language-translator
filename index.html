🎓 How to use this platform

1. Select your country's sign language at the top. Each country has its own independent sign language — BSL and ASL are completely different despite both being from English-speaking countries.

2. Start the camera and position your hand clearly in the frame. Good lighting and a plain background give the best results.

3. Hold each sign steadily for about 1–2 seconds. The system smooths detections over 5 frames to avoid flickering.

4. Use the Alphabet tab to practice each letter. Use the Numbers tab to learn how large numbers are signed. Use Phrases for real conversations.

🧠 Train your own model

1Go to teachablemachine.withgoogle.com → New Project → Pose Project
2Create a class per sign, record 30–50 webcam samples each
3Train → Export → TensorFlow.js → Upload → copy shareable URL
4Paste URL below and click Load — your model activates instantly

🏭 Production backend (Option C)

1Dataset: WLASL (2,000 ASL words) — github.com/dxli94/WLASL
2Extract: Run MediaPipe on each video frame → get landmark sequences (63 values × N frames)
3Train: LSTM model on sequences → sign label with confidence
4Serve: Flask API on Render → this app POSTs landmarks → API returns sign
